Operational Dashboard

The dashboard is where site teams review, investigate, and report on structured events generated by the platform. All operations work from time-stamped event data — not raw footage.

Review

  • Monitor event timelines
  • Filter by camera, zone, time, event type

Investigate

  • Drill into specific incidents
  • View cross-camera context
  • Track movement across zones

Report

  • Generate shift summaries
  • Produce incident reports
  • Export compliance records (PDF / CSV / HTML)

Workflow example

Restricted Zone Incident

1.

A person enters a restricted zone. The platform logs an access violation event with timestamp, camera ID, and zone reference.

2.

The operator opens the dashboard, filters the timeline to the relevant zone and time window, and reviews the sequence of events.

3.

A structured incident report is generated — including event timeline, location data, and linked footage reference — ready for review or export.

Event Timeline & Search

All site activity appears as structured, time-stamped records. Operators search by time range, camera, zone, event type, or specific conditions.

  • Chronological event sequence with full context
  • Cross-camera queries to track activity across zones
  • No raw footage scrubbing required

Structured Reporting

Reports are generated directly from the event record. Outputs are consistent, auditable, and available in multiple formats.
