Operational Dashboard

The dashboard is the control layer of the platform. It provides real-time visibility of site activity while maintaining a structured, searchable record for review, reporting, and oversight.

Live events are monitored as they happen. Structured records are stored automatically, allowing operators to respond immediately and generate formal outputs when required.

All features operate on structured event data — not raw video playback.

Live Operations

Real-time feed

Live event stream as activity is detected across monitored zones.

Rule triggers

Active alerts when defined conditions are met or violated.

Zone monitoring

Track activity and conditions across defined site areas.

Operator response

Immediate visibility for on-duty teams to act on live events.

Review, Reports & Investigation

All operational outputs are generated from the structured event record. Review activity, generate reports, and investigate incidents from factual, time-stamped data.

Timeline Review

  • • Chronological event sequences by camera, zone, or time
  • • Incident context with full event history
  • • Activity summaries by hour, shift, or day

Structured Reporting

  • • Shift handover summaries
  • • Incident reports with timeline and evidence
  • • Compliance and audit records
  • • Export as HTML, PDF, or CSV

Event Search

  • • Query by time range, location, or event type
  • • Historical investigation over days, weeks, or months
  • • Pattern and anomaly detection
